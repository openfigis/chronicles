Welcome to the  chronicles webservice application
 

Results are cached, maximum number of objects in the cache is 500.
If you hit an operation for the first time, you may have to wait a few minutes.

This is the a REST based webservice for Chronicles operations. Go here to for more documentation or ask erik.vaningen@fao.org


Chronicles operation Syntax Example
generate development chart chart/development/startyear/{startyear}/endyear/{endyear}/clusters/{clusters}/top/{top}/chart.png chart/development/startyear/1950/endyear/2007/clusters/10/top/200/chart.png
chart/development/startyear/{startyear}/endyear/{endyear}/clusters/{clusters}/top/{top}/un/{un}/chart.png
*1)		 chart/development/startyear/1950/endyear/2007/clusters/10/top/200/un/392/chart.png 
generate capture chart chart/capture/startyear/{startyear}/endyear/{endyear}/top/{top}/chart.png chart/capture/startyear/1950/endyear/2007/top/200/chart.png
chart/capture/startyear/{startyear}/endyear/{endyear}/top/{top}/un/{un}/chart.png
*1)		 chart/capture/startyear/1950/endyear/2007/top/200/un/392032/chart.png
generate development
csv		 csv/development/track/{track}/startyear/{startyear}/endyear/{endyear}/clusters/{clusters}/top/{top}
*2)		 csv/development/track/3/startyear/1950/endyear/2007/clusters/10/top/200/development.csv
csv/development/track/{track}/startyear/{startyear}/endyear/{endyear}/clusters/{clusters}/top/{top}/un/{un}
*2)		 csv/development/track/3/startyear/1950/endyear/2007/clusters/10/top/200/un/156/development.csv
generate capture csv csv/capture/track/{track}/startyear/{startyear}/endyear/{endyear}/top/{top}
*2)		 csv/capture/track/1/startyear/1995/endyear/1999/top/10/capture.csv
csv/capture/track/{track}/startyear/{startyear}/endyear/{endyear}/top/{top}/un/{un}
*2)		 csv/capture/track/1/startyear/1995/endyear/1999/top/10/un/352/capture.csv

*1) The parameter un can contain one or more UN country codes. Go here for the list of UN country codes.  Example for Japan is 392. Example for Japan and Argentia is 392032
*2) Go here for an explanation of the tracks
